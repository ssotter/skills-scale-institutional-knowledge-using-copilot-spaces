## When to use
Use this template whenever introducing significant changes to processes, tools, workflows, or organizational structures that affect the project team or stakeholders.

## Checklist for Change Manager

### Before Implementation
- [ ] Change impact assessment completed
- [ ] Stakeholder analysis and readiness assessment done
- [ ] Communication plan developed and approved
- [ ] Training materials prepared
- [ ] Change champions identified and engaged
- [ ] Risk mitigation strategies in place
- [ ] Go-live plan documented and reviewed

### During Implementation
- [ ] Communications sent per plan
- [ ] Training delivered to all affected groups
- [ ] Support resources available and visible
- [ ] Issues tracked and resolved quickly
- [ ] Regular updates provided to stakeholders
- [ ] Success metrics being monitored

### After Implementation
- [ ] Adoption metrics reviewed regularly
- [ ] Feedback collected and acted upon
- [ ] Lessons learned session conducted
- [ ] Documentation updated
- [ ] Change integrated into standard processes
- [ ] Final report to sponsor and stakeholders
