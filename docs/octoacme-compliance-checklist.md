## Escalation Path

**Compliance concerns should be escalated as follows**:
- **Level 1** (Minor issues): Compliance Analyst → Project Manager
- **Level 2** (Moderate risk): Compliance Analyst → Legal/Security Teams
- **Level 3** (High risk/blocker): Compliance Analyst → Legal/Security → Executive Sponsor

**For immediate compliance incidents**: Contact Security on-call immediately and follow incident response procedures.
